A responsive web app for pet adoption, built using plain JavaScript and Tailwind CSS. Created as part of a resit for Semester Project 2.

## Overview

A pet adoption organization needed a simple platform where:

- **Users** can browse and search for available pets  
- **Admins** can log in to manage pet listings (add, edit, delete)

## Tech

### Vanilla JS

- No frameworks, so everything is built from scratch  
- Great for learning core web development  
- Full control over how the app works
- Easy to read

### Hash Based Routing

- Enables simple singlepage navigation    
- Works reliably across all browsers

### Tailwind CSS

- Fast styling with utility classes  
- Built in mobile responsiveness  
- Keeps the design clean and consistent

## Getting Started

1. **Install dependencies**

   ```bash
   npm install
````

2. **Start the development server**

   ```bash
   npm run dev
   ```

3. **Open the link**

   ```bash
   http://localhost:5173
   ```

## Features

* **Pet Listings:** Browse pets with search and pagination
* **Pet Details:** View detailed info about each animal
* **Admin Panel:** Log in to add, edit, or remove pet listings
* **User Authentication:** Simple user registration and login
* **Responsive Design:** Optimized for mobile, tablet and desktop
* **Search Functionality:** Filter pets by name, species, or breed
